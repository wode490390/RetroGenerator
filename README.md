## Configurations

### config.yml
```yaml
line:
  # Block ID used to generate the line
  material: 236
  meta: 5
background:
  # Block ID used to generate the background
  material: 236
  meta: 15
```

## Compiling
1. Install [Maven](https://maven.apache.org/).
2. Run `mvn clean package`. The compiled JAR can be found in the `target/` directory.
